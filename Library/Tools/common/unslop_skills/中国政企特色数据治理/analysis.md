# 分析报告：中国政企特色数据治理文章中的“AI味”规律

通过对多篇由AI生成（模拟）的政企数据治理文章进行交叉对比，我们发现了明显的结构性塌陷与词汇同质化现象（Slop）。如果不对其进行约束，AI极易掉入八股文的窠臼，导致生成的文章缺乏真正的见地，变成空洞的辞藻堆砌。

## 1. 标题规律 (Title Patterns)
**高频特征：** 倾向于使用大词和四平八稳的公文格式缩写。
- 固定搭配：“关于...的思考”、“...体系建设”、“构建...新格局”。
- 喜欢在标题中强行加注：“数字化转型中”、“新时代下”。

## 2. 开篇与引入 (Openings)
**高频特征：** 100% 采用宏观背景作为起手式。
- “随着数字经济/大数据战略的飞速发展/深入推进……”
- “在当今数字化浪潮之下……”
- 将数据强行比喻为：“新型生产要素”、“核心驱动力”、“必由之路”。

## 3. 过渡与逻辑连接 (Transitions & Structure)
**高频特征：** 极端依赖生硬的枚举式过渡，缺乏自然的段落承接。
- “一方面……另一方面……”
- “首先……其次……再次……”
- “一是坚持统筹规划……二是聚焦业务场景……”

## 4. 高频职场/政务“黑话”词汇 (Clichés & Buzzwords)
**高频特征：** AI 抓取了大量政企PPT常用词组，并在文中密集释放，导致“空心化”。
- **动词类**：“赋能”、“打破”、“打通”、“落地生效”、“统筹”、“夯实”。
- **名词/形容词类**：“信息孤岛”、“数据烟囱”、“部门壁垒”、“顶层设计”、“一把手工程”、“抓手”、“闭环”、“双刃剑”、“数据底座”、“全生命周期”。
- **特定黑话搭配**：“小切口带动大破局”、“脱实向虚”、“可用不可见，相逢不相识”。

## 5. 结尾与展望 (Endings)
**高频特征：** 结尾句式高度趋同，充满无意义的美好愿景。
- “总而言之”、“综上所述”、“展望未来”。
- “……不是一蹴而就的，是一场持久战。”
- “必将谱写……高质量发展的崭新篇章”、“注入源源不断的强劲动力”、“迎来……新生态”。

---
**结论**：AI在处理“政企数据治理”这个主题时，为了追求“专业感”和“官方感”，会过度拟合公文材料中的宏大叙事和特定互联网黑话，导致文章脱离具体的业务痛点，变成四平八稳的“正确废话”。必须通过 `skill.md` 在系统层面彻底屏蔽这些特定的叙事框架和陈词滥调。
